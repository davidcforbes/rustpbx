The tracking code enables dynamic number insertion on your website. When a visitor arrives, the tracking code replaces your target numbers with tracking numbers, allowing you to attribute phone calls to the correct advertising source.

Important Setup Notes

  • ● Avoid conflicts with other call tracking scripts on the same page. Only one tracking code should be active per page.
  • ● Hardcoded phone numbers on your website must exactly match the target numbers configured in your account.
  • ● Phone numbers inside embedded elements (iframes) require separate tracking code installation within the iframe source.
